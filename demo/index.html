Easy toast notification

Lightweight library for toast notifications. Written in vanilla javascript and does not require other dependencies. Try the pop-up notifications on this page.

Github Source NPM

Config properties

  • duration - In seconds
  • type - Notification style
  • hasCloseButton - Show close button
  • showTimeline - Show timeline (progress bar)
  • autoclose - Close notification when timer ends
  • stopOnHover - Pause the timer when you hover over a notification
  • events - Object contains two available event: click and closeClick
